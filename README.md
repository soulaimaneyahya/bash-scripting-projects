**Caching Eviction Policies**
Listed below are major cache eviction strategies:

**LRU Algorithm** = Least Recently Used memory // cache
Stores // limited items; evicts the least recently accessed item when full

**LFU Algorithm (Least Frequently Used)**

**FIFO (First-In First-Out)**
Evicts the _oldest inserted_ item first.

**MRU (Most Recently Used)**
Evicts the _most recently accessed_ item.
